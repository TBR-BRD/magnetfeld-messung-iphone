Magnetfeld-Messung

Webapp zum Testen, ob ein Smartphone beim Passieren eines Eingangs magnetische Störungen erkennt. Auf dem iPhone ist das ein Kompass-/Orientierungs-Fallback, keine garantierte Rohmessung in µT.

Secure Context wird geprüft… Sensor-Support wird geprüft… lokal, kein Upload
Starte die Messung über den Button. Auf iOS muss die Sensorfreigabe aus einer echten Nutzeraktion kommen.

Kurzanleitung für den Test an Warensicherungs-Gates
  1. Webapp über HTTPS öffnen und auf Messung starten tippen.
  2. Sensorzugriff erlauben. Beim iPhone kann die Abfrage „Bewegung & Ausrichtung“ heißen.
  3. Ein paar Meter vom Gate entfernt das iPhone ruhig halten und Nullpunkt setzen drücken.
  4. Langsam durch das Gate gehen, möglichst ohne das iPhone zu drehen.
  5. Danach CSV herunterladen und im Diagramm nach Peaks suchen.
Warum zeigt mein iPhone keine X/Y/Z-Werte?

iOS Safari stellt in normalen Webseiten meist keinen Roh-Magnetometer als Generic Sensor API bereit. Diese App nutzt dann Kompass- und Orientierungswerte als indirekten Indikator. Für echte µT-Rohdaten auf dem iPhone wäre eine native iOS-App mit Core Motion besser.

Datenschutz

Diese Demo sendet nichts an einen Server. Alle Messwerte bleiben im Arbeitsspeicher des Browsers, bis du sie als CSV herunterlädst oder die Seite schließt.

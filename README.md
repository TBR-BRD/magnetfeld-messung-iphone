# Magnetfeld-Messung iPhone

Lokale Webapp für Magnetfeld-/Kompassmessung im Browser.

## Start

```bash
python3 -m http.server 8080
```

Für iPhone-Sensoren: HTTPS nutzen.

## Privacy

Keine Serverübertragung. CSV bleibt lokal.
